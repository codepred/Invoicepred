## 🚀 Installation Guide

### ✅ Install Dependencies
To install the necessary packages, run:

```bash
npm install
```

### 💪 Build for Production
To build the application for production, use:

```bash
npm run build
```

### 🏠 Run Locally
To start the application in development mode, execute:

```bash
npm run dev
```
